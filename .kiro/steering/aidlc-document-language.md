# AI-DLC 문서 언어 정책

## 기본 언어
- 사용자에게 제공하는 응답과 새로 생성하거나 갱신하는 문서의 자연어는 한국어로 작성한다.
- 기존 문서의 영어 자연어 제목과 본문도 수정 범위에 포함될 때 한국어 중심으로 갱신한다.

## 번역하지 않는 항목
- 코드, 명령, 파일 경로, URL, Mermaid 구문은 원문을 보존한다.
- 기술 식별자, 계약명, 타입명, 상태값, 이벤트명, 요구사항·규칙·스토리 ID는 원문을 보존한다.
- `audit.md`의 `User Input`과 `AI Prompt`는 감사 원문이므로 번역하거나 수정하지 않는다.

## Kiro Spec 호환성
- Kiro Spec 진단이 정확한 영문 스키마 제목을 요구하면 한국어 사용자 제목을 먼저 두고 영문 제목을 독립된 별칭으로 함께 유지한다.
- 영문 스키마 별칭은 진단 호환 목적이며 나머지 설명과 콘텐츠는 한국어로 작성한다.

## 적용 범위
- 이 정책은 `aidlc-docs/`와 AI-DLC 관련 사용자 응답에 기본 적용한다.
- 외부 계약 또는 도구가 영문 고정값을 요구하는 경우 해당 고정값만 원문으로 유지한다.
